Описание структуры данных:

Класс Item - класс товара, который содержит название товара и ассоциативный массив его компонентов (ключ - компонент, значение - нужное количество)

Класс Country - класс страны, который содержит название страны и массив товаров (со всей присущей им информацией, поданной в виде структуры data)

Структура Data - вспомогательная структура для массива товаров страны, хранит указатель на товар и массив записей статистики

Структура Records - структура, для хранения статистики по товарам для разных периодов (хранит год/количество экспорта/импорта/производства)
